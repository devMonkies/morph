Morph UI

(Documentación de la version 2.0 en progreso)
¿Qué es Morph UI?

Morph UI es un Sistema de Diseño web hecho en CSS que permite desarrollar componentes para Interfaces Web.

A través de una guía de diseño, pautas de usabilidad (evaluación heurística), principios de legibilidad y extensibilidad de codigo.

También es una base en CSS que permite programar la interacción de componentes web con cualquier framework o biblioteca en JS (React, Angular, Vue)

Características
  • Clases en CSS en base a una semántica
  • Composición (respetar la prioridad de las etiquetas HTML por su nombre y propósito al escribir CSS)
  • Modelo de caja con proporción y simetría
  • Paleta de colores en base a un sistema primario y secundario para los componentes
  • Guia de estilos basada en los estandares de la web
